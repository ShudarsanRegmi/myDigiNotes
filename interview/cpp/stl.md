# Unordered Map

```cpp
unordered_set<int> s;
s.insert(5);
s.erase(5);
s.count(2); // returns true if count matches
s.clear();
s.size();
s.empty();

s.begin();
s.end();
```


# Vector
```cpp
v.begin();
v.end();
v.rbegin(); // end
v.rend(); // beginning

v.push_back(5); // append
v.insert(it, 5); // stores 5 at the place where iterator points

v.back();
v.front();

v.clear();
v.empty();
v.size();


v.max_size();
v.capacity();
```
