## MongoDB Basics to Advanced Concept

- Database, collection, documents
- JSON like Documents are stored as BSON
- No rigid structure, eliminating the need of disruptive migrations.   
- mongodb provides flexibility, but disciplie is expected at application layer.

- Application interacts with mongodb through drivers, which translates queries into database operations.
- Internally, MongoDB handles indexing, storage engines, and concurrency control.
