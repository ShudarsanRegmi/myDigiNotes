# Java Lab : 01


### Creating class and object
```java
public class Student {
    int id = 5;
    String name = "hari";
    public static void main(String[] args) {
        Student s = new Student();
        System.out.println(s.id);
        System.out.println(s.name);
    }
}

```
#### Key notes:
- main is defined within the class
